# ⚡ Electro Modus

Сайт компании, занимающейся электромонтажом, ремонтом и модернизацией электроники.

---

## 📌 О проекте

Electro Modus — это учебный проект с базовой версткой:

* семантический HTML
* структура по БЭМ
* layout через CSS Grid
* базовая стилизация

---

## 🚀 Функционал

* 📂 боковое меню (sidebar)
* 📄 несколько секций (главная, о нас, услуги, контакты)
* 📱 адаптивная основа (viewport)
* ⚡ подключение CSS и JS

---

## 🛠️ Технологии

* HTML5
* CSS3
* JavaScript (базовый)

---

## 📁 Структура проекта

```
project/
```
